# Secure Network Merger Design (Company A & B)

## 📌 Overview
This project presents a secure network merger plan between a financial services company and a medical software company.

The design focuses on:
- Zero Trust Architecture
- Cloud Integration (Microsoft Azure)
- Risk Assessment (NIST SP 800-30)
- Regulatory Compliance (PCI DSS & HIPAA)

## 🔐 Key Security Features
- Multi-Factor Authentication (MFA)
- Network Segmentation
- Secure Vendor Access Controls
- Azure VPN Gateway for encrypted communication

## ⚠️ Identified Risks
- Outdated servers
- Weak remote authentication
- Default credentials
- Unsecured vendor access

## 🛠️ Technologies Used
- Microsoft Azure
- VPN Gateway
- Firewall & Network Segmentation
- Zero Trust Security Model

## 💰 Budget Consideration
Solution designed under $50,000 using:
- Existing infrastructure reuse
- Cloud-based security tools
